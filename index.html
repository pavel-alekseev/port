Story

I am freelance designer with five years of experience in various fields. I studied graphic design for two years in Latvian design academy, then worked full time as print designer for few years and studied UI design. Print design was a good foundation that allowed me to start working in other fields.

Now I mainly design web apps and closely work with developer teams, I do print and branding projects from time to time to keep work diverse. As a freelancer, I can pick my own tools and design methods, so I experiment with process, I try to refine it, create my own libraries to produce design faster.

Project task
Design user interface for energy managment system app and website.

Product description
Sunplug is all in one solar energy system, along side all needed solar panel components it provides app that informs users on systems energy state. App monitors all nessesary information, like energy consumption, generation, import and export to grid. Every few months product gets new features.

Web app
Web app made based on ionic framework provides energy related data to user in realtime.

Next

Sunplug for tablet
Tablet interface was created so that it could have all nesessary features on one screen. Component styling kept the same as on mobile to ensure visual consistentsy. Similar to mobile, interface was designed for vertical and horizontal screen orientation.

Landing page
Website for sunplug, contains several charts and 3d model created by me to better explain key system features to users.
